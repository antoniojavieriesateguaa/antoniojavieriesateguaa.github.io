Ejercicio 2

Indica para que sirven estas etiquetas:

Details: es usado como un widget de revelación a través del cual el usuario puede obtener información adicional

Mark: sirve para representar un texto relevante, este puede ser importante o no dentro del contexto de la página

Summary: representa un resumen, título o etiqueta para los contenidos de un elemento details

Hr: se trata de una división temática o quiebre en el texto
